## To set up the python environment use below commands
1 mkdir env ( move inside the env folder)
2 mkdir venv 
3 command:- python -m venv venv
4 source env/venv/Scripts/activate (on git terminal in vs code)
5 env/venv/Scripts/activate (on windows)
6 source ev/venv/bin/activate (on mac)
7 install requirements. Command:- pip install -r requirements.txt
