## To set up the python environment use below commands \n
1 mkdir env ( move inside the env folder) \n
2 mkdir venv \n
3 command:- python -m venv venv \n
4 source env/venv/Scripts/activate (on git terminal in vs code) \n
5 env/venv/Scripts/activate (on windows) \n
6 source env/venv/bin/activate (on mac) \n
7 install requirements. Command:- pip install -r requirements.txt
